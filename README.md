# Explorify 🌍✈️

Explorify is a full-stack travel planning platform that helps users plan their entire trip from a single website.

## Features

* 🤖 AI-Powered Trip Planner
* 💰 Budget Estimation for Trips
* 🗺️ Interactive Maps for Navigation
* 🏨 Hotel Booking
* ✈️ Flight Booking
* 🚌 Bus Ticket Booking
* 📅 Day-wise Travel Itinerary
* 👥 Share Trip Plans with Friends
* 📍 Destination Recommendations Based on Budget
* 📱 Responsive and Modern UI

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### APIs & Services

* Google Maps API
* AI Travel Recommendation System
* Hotel & Flight Booking APIs

## Installation

```bash
git clone https://github.com/rajputadarsh12/Explorify.git
cd Explorify
npm install
npm start
```
